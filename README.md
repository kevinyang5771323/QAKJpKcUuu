# 排班管理系统简介 java403

## 项目概述

排班管理系统是基于Flask和Vue.js的前后端分离架构，旨在提供灵活、易用的排班管理解决方案。

## 技术栈

### 后端

- Flask
- Flask-SQLAlchemy
- Flask-RESTful
- Flask-Redis
- MySQL
- Redis

### 前端

- Vue.js
- Element UI
- ECharts

## 功能模块

- **排班计划管理**
  - 周排班
  - 月排班
  - 排班计划导出
- **员工管理**
  - 员工信息导入
  - 员工信息修改（补偿排班计划）
  - 员工信息删除
- **值班地点管理**
  - 添加
  - 删除
  - 修改
- **节假日管理**
  - 少数民族节假日添加
  - 删除
  - 修改
- **人员状态管理**
  - 添加
  - 删除
  - 修改
- **人员角色管理**
  - 添加
  - 删除
  - 修改
- **统计与分析**
  - 人员状态分析
  - 人员预排班次数分析
  - 人员状态百分比分析

## 系统角色

- 管理员：负责排班计划的管理与维护，员工信息的管理，以及系统设置。
- 用户：查看个人排班信息，更新个人状态。

## 运行环境

- Python 3.10
- MySQL
- PyCharm
- Node.js v14.15.0
- HBuilder

## 部署步骤

1. 准备后端所需Python环境及数据库。
2. 部署后端服务，利用提供的CentOS镜像在虚拟机上启动。
3. 构建前端项目，部署至Web服务器。

## 目录结构

```
/backend      # 后端代码目录
/frontend     # 前端代码目录
```

## 核心亮点

- 前后端分离，易于维护和扩展。
- 使用Flask-SQLAlchemy和Redis进行数据持久化，确保数据存取的高效性。
- 提供详尽的统计与分析功能，助力管理者高效决策。
